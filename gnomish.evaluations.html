Gnomish::Evaluations

Subroutines for selection from populations.


computeTraitPhenotypeMean

Compute the phenotypic mean value for a particular trait. Make sure that the phenotypic values for the particular TraitName has been computed first!

Parameters:

  • Population (reference)

  • String: TraitName

Example:

    computeTraitPhenotypeMean ( Population => \$p,
                                TraitName => 'Height');


computeTraitPhenotypeVariance

Compute the phenotypic variance for the named trait.

Parameters:

  • Population (reference)

  • String: TraitName

Example:

    computeTraitPhenotypeVariance ( Population => \$p,
                                    TraitName => 'Height' );


selectGenotypeArray

Select genotypes from a population with a particular locus that at the specified locus. Return an array with the usual suspects. Note that the returned members are copies of the original genotypes, and do not belong to any particular population.

Parameters:

  • Population (reference)

  • Int: Locus

  • String: LocusState

Example:

    selectGenotypeArray ( Population => \$p
                          Locus => 45,
                          LocusState => "AC" );

countAlleleStatesInGenotype

Count the number of different allele states in the allele strings of the genotype. Return the result as a hash object.

Parameters:

  • Genotype: Genotype (reference to a genotype object)

Example:

    my %allele_states = countAlleleStatesInGenotype (
                                  Genotype => \$genotype );



getAlleleStateCountInGenotype

Count the number of times a particular allele states in a genotype.

Parameters:

  • Genotype: Genotype (reference to a genotype object)

  • String: AlleleState

Example:

    my $count = getAlleleStateCountInGenotype (
                            Genotype => \$genotype,
                            AlleleState => "aa" );



countAlleleStatesInPopulation

Count the occurrences of all allele states in the population. Return the results in a hash object.

Parameters:

  • Population: Population (reference to a population object)

Example:

    my %allele_states = countAlleleStatesInPopulation (
                            Population => \$population );



getAllleStateCountInPopulation

Get the count of the number of times a particular allele state occurs in the genotypes in the population.

Parameters:

  • Population: Population (reference to a population object)

  • String: AlleleState

Example:

    my $count = getAlleleStateCountInPopulation (
                                Population => \$population,
                                AlleleState => "aa" );
