Gnomish::Population

This module incorporates both the class definition for the Population class and regular functions for operations on population objects.


Functions that operate on Population Objects

These functions exist external to the Population class definitions.


doubledHaploids

Make a new population from a parent population, where one or more doubled haploid progeny is created.

Parameters:

  • Int: NProgeny (number of progeny to create per parent in the new population)

  • Population: ParentPop (reference to the parent population -- this population must have at least one member)

Example:

    my $parent_pop = new Gnomish::Population;
    $parent_pop->makePopulation ( NMembers => 1,
                                  CreateMethod => "RANDOM_ALLELE",
                                  Genome => $genome );

    my $new_pop = doubledHaploids ( NProgeny => 100,
                                    ParentPop => \$parent_pop );



selfedProgeny

Create a new population in which each parent in the parent population is crossed with itself one or more times.

Parameters:

  • Int: NProgeny (number of progeny to create per parent in the new population)

  • Population: ParentPop (reference to the parent population -- this population must have at least one member)

Example:

    my $new_pop = selfedProgeny ( NProgeny => 100,
                                  ParentPop => \$parent_pop );



clonalProgeny

Create a new population in which each parent is cloned one or more times.

Parameters:

  • Int: NProgeny (number of progeny to create per parent in the new population)

  • Population: ParentPop (reference to the parent population -- this population must have at least one member)

Example:

    my $new_pop = clonalProgeny ( NProgeny => 100,
                                  ParentPop => \$parent_pop );


singleSeedDescent

Perform N generations of single seed descent. This function creates one selfed progengy for each member of the parent population (F1 generation). Then each member in the new population is replaced by one of its selfed progeny. The process is repeated until N generations have been reached.

Parameters:

  • Int: NGenerations (number of generations)

  • Population: ParentPop (reference to the parent population -- this population must have at least one member)

Example:

    my $new_pop = singleSeedDescent ( NGenerations => 20,
                                      ParentPop => \$parent_pop );


randomMate

Randomly mate genotypes in the population, creating new population. Take existing population, make a new population by crossing random parents from the existing population. Each cross results in one new member of the the new population.

Note that parent selection is completely random, that is a parent can be crossed with itself, or selected for crossing more than once. If there is only one parent individual in the parent population, the result is a selfed population of N progenies.

Parameters:

  • Int: NProgeny (number of progenies in the new population)

  • Population: ParentPop (reference to the parent population -- this population must have at least one member)

Example:

    my $new_pop = randomMate ( NProgeny => 100,
                               ParentPop => \$parent_pop );


chainCrosses

Create a new population by sequentially crossing members of the population. This procedure crosses the first individual with the second, the second with the third, etc. The last individual is crossed with the first so that the number of crosses made is equal to the number of individuals in the parental population. One progeny is made from each cross.

Parameters:

  • Population: ParentPop (reference to the parent population -- this population must have at least one member)

Example:

    my $new_pop = chainCrosses ( ParentPop => \$parent_pop );



fullDiallel

Create a new population in which every individual is crossed with every others in the population, including itself. This function is essentially a version of 'NC_DesignII' (see below), but on a single population.

Parameters:

  • Population: ParentPop (reference to the parent population -- this population must have at least one member)

Example:

    my $new_pop = fullDiallel ( ParentPop => \$parent_pop );



NC_DesignII

This mates all individuals in one population with all members of another population, making one progeny from each cross.

Parameters:

  • Population: ParentPop1 (reference to a parent population -- this population must have at least one member)

  • Population: ParentPop2 (reference to a parent population -- this population must have at least one member)

Example:

    my $new_pop = NC_DesignII ( ParentPop1 => \$p1,
                                ParentPop2 => \$p2 );


backCrosses

This performs crosses between all individuals in one population (the donor population) and a single individual (the recurent parent) in a second population. One or more progenies can be produced from each cross.

Parameters:

  • Genotype: RecurentParent (reference to the recurrent parent)

  • Population: DonorPop (reference to the donor population -- this population must have at least one member)

Example:

    my $new_pop = backCrosses ( RecurrentParent => \$genotype,
                                DonorPop => \$donor_population );


mergePopulations

Merge two specified populations, return a new population object with the two parent population merged.

Parameters:

  • Population: Pop1 (reference to a population object)

  • Population: Pop2 (reference to a population object)

Example:

    my $merged_pop = mergePopulation ( Pop1 => \$p1,
                                       Pop2 => \$p2 );



makeRandomAllelePopulation

Create a random population from scratch with random alleles. With this creation method, a random allele is chosen and assigned to each locus on each allele string in the genotype.

Parameters:

  • Int: NMembers (number of genotypes in the new population)

  • Genome: Genome (genome that the population will belong to)

Example:

    my $population = makeRandomAllelePopulation ( NMembers => 100,
                                                  Genome => $genome );



makeRandomHomozygousPopulation

Create a new population with randomly chosen homozygous alleles. In this case, at each locus, a random allele is chosen. This allele is then assigned to both allele strings (gametes) in the genotype.

Parameters:

  • Int: NMembers (number of genotypes in the new population)

  • Genome: Genome (genome that the population will belong to)

Example:

    my $population = makeRandomHomozygousPopulation ( NMembers => 100,
                                                      Genome => $genome );



makeHomozygousPopulation

Create a new population with alleles homozygous to the specified allele.

Parameters:

  • Int: NMembers (number of genotypes in the new population)

  • Genome: Genome (genome that the population belongs to)

  • Char: Allele (allele to assign to the genotypes in the population)

Example:

    my $population = makeHomozygousPopulation ( NMembers => 100,
                                                Genome => $genome,
                                                Allele => "a" );


Population Class methods

makePopulation

Create new population and populate using the specified create method.

Parameters

  • String: Name (optional)

  • Int: NMembers (number of genotypes to create in the population)

  • CreateMethod (creation rule used to create new genotypes in the population. Please refer to the documentation for Gnomish::Genotype::makeGenotype)

  • Genome: Genome (genome that members of the population belong to)

Example:

    my $pop = new Gnomish::Population;
    $pop->makePopulation ( Name => "ExamplePopulation",
                           NMembers => 100,
                           CreateMethod => "RANDOM_ALLELE",
                           Genome => $genome );



setName

Set the population name.

Parameters:

  • String: Name

Example:

    $pop->setName ( Name => "NewPopulationName" );



getName

Return the name of the population.

Parameters: none

Example:

    my $pop_name = $pop->getName();



nMembers

Return the number of members in the population.

Parameters: none

Example:

    my $n = $pop->nMembers();



addMember

Add a new genotype to the population.

Parameters:

  • Genotype: Genotype (reference to a genotype that is being added to the population)

Example:

    $pop->addMember ( Genotype => \$genotype );



getMemberRef

Return a reference to a particular member of the population.

Parameters:

  • Int: Member (index of a particular member of the population)

Example:

    my $genotype_ref = $pop->getMemberRef ( Member => 10 );



getMember

Return a copy of a particular member of the population.

Parameters:

  • Int: Member (index of a particular member of the population)

Example:

    my $genotype = $pop->getMember ( Member => 10 );



shuffle

Randomly shuffle the members in the population.

Parameters: none

Example:

    $pop->shutffle();



computePhenotypes

Compute the phenotypic values for each genotype in the population. (This method iterates over every genotype in the population and calls the genotype's computePhenotypicValue method. Please refer to the documentation for Gnomish::Genotype::computePhenotypicValue for more information.)

Parameters: none

Example:

    $pop->computePhenotypes();
