Gnomish::Stdlib

Miscellaneous functions that do not fit anywhere else.



initialize

Initialize the random number generator. Remember to execute this function before doing any other work with Gnomish!

Parameters: none

Example:

    require Gnomish::Stdlib;
    initialize();


gaussianRand

Compute a random number according to a Gaussian distribution.

Parameters:

  • Float: Variance (variance of the gaussian distribution)

Example:

    my $r = gaussianRand ( Variance => 3.5 );



computeAlleleStateSet

The 'AlleleSet' is an array that contains the valid allele values in the genome. For example, if an allele can have a value of 0, 1, 2, or 3 then AlleleSet is (0, 1, 2, 3). This function computes the various pairwise permutations that are possible and their ordinate value. For example, the set 0, 1, 2, 3 can have pairwise relationships like: 00, 01, 02, 03, 10, 11, 12, 13, 20, 21, 22, 23, ..... However, 10 is the same as 01, 20 is the same as 02, etc. This function returns a look up table that maps the ordinate value of a pair. (This is used by internal Gnomish routines).

Parameters:

  • Array: AlleleSet

Example:

    my @allele_set = ( 'a', 'b', 'c', 'd');
    my %allele_states = computeAlleleStateSet ( AlleleSet => \@allele_set );
